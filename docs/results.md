# Results

No real-data participant-level predictions or real word-level feature tables are included in this release candidate. The bundled synthetic example writes aggregate metrics to `metrics.json` and one cross-fitted profile row per synthetic reader.

Any real-data result must be regenerated from externally obtained CopCo data with reader-disjoint evaluation and fold-local residualization. Public reporting should use aggregate metrics only and must keep the reader-level target clear.
