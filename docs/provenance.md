# Provenance

This public release candidate was built from local research-code evidence and prior audit notes. Public files were rewritten under the `samo_copco` package name to avoid private paths, internal method labels, raw data, participant rows, and local audit artifacts.

The source logic basis is limited to public-safe behavior: schema validation, reader-disjoint splits, fold-local residualization, reader-profile aggregation, and aggregate metric reporting.

Repository code license: `MIT`. CopCo data are not redistributed in this repository, and users must obtain CopCo separately through the appropriate access route.
